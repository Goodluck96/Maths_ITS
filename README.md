# Intelligent Tutoring System (ITS)

This project implements an ontology-driven Intelligent Tutoring System for mathematics learning.
It integrates a Protégé OWL ontology with a Python-based adaptive tutor, a Tkinter graphical user
interface, and a Matplotlib mastery dashboard.

## Features
- Formal OWL ontology for knowledge representation
- Adaptive skill and difficulty selection
- Real-time mastery tracking
- Tkinter GUI with embedded visual analytics
- Explainable feedback and hints

## Technologies Used
- Python 3
- Tkinter
- Matplotlib
- Protégé (OWL ontology)
